Hinweise zur Webseite

Problematik

Eventuell kann es zu Aktualisierungsproblemen der Webseite führen. Um dieses Problem zu lösen, hilft es den Browser cache (Browserverlauf) zu leeren, damit die Webseite neu geladen werden kann. Unten im Impressum wird angegeben, wann die Webseite zuletzt aktualisiert wurde. Wenn dort ein neueres Datum steht, aber keine neuen Inhalte verfügbbar sind, dann gibt es ein Aktualisierungsproblem und die Webseite muss neu geladen werden.

Hinweise zur Software

Problematik

Wenn Sie eine meiner Software downloaden möchten, kann es passieren, dass Google Drive in einem neuen Fenster folgende Meldung anzeigt:
"Google Drive kann keinen Virenscan für diese Datei durchführen.". Das liegt daran, das ich diese Webseite und ihre Inhalte, also die zum download verfügbare Software, über Google Drive verwalte. Damit ist lediglich gemeint, dass Google Drive nur Dateien bis zu einer bestimmen Datenmenge auf Viren scannen kann. Allerdings überschreitet meine Software zum Teil diese Datenmenge und somit ist kein interner Scann von Google Drive möglich. Meine komplette Software ist frei von Viren und mit bestem Gewissen erstellt worden. Es werden keine zusätzlichen Daten gespeichert oder ausgewertet.
Wenn Sie dennoch bedenken haben, empfehle ich Ihnen folgende Webseite, auf der Sie ganz leicht die JAR- beziehungsweise EXE-Datei hochladen können und kostenfrei, ohne sich anmelden zu müssen, die komplette Datei auf Viren prüfen lassen können: www.virustotal.com. Dadurch erhalten Sie eine Bestätigung das meine Software keine bösen Absichten hat und fälschlicher Weise, lediglich zur Sicherheit, als Virus erkannt wird.
Falls weiterhin Fragen oder Bedenken zu meiner Software bestehen, können Sie mich gerne unter code-eagle@outlook.de kontaktieren.

Download

Wenn Sie auf einen Downloadknopf meiner Webseite klicken werden Sie eventuell zu einem neu geöffneten Fenster geleitet, indem Sie "Trotzdem herunterladen" anklicken müssen. Eventuell erscheint dann nochmals eine Abfrage vom Browser, ob dieser mehrerer Dateien, also einen Ordner, einmalig downloaden darf. Erst wenn dies bestätigt wurde, beginnt der Download.

Nutzung

Wenn Sie die Software erfolgreich heruntergeladen haben, begeben Sie sich in den Ordner, wo die Datei gespeichert wurde. Standardmäßig ist dies der Downloadordner. Es liegt nun eine ZIP-Datei (der Ordner mit dem Reißverschluss) vor, um die Datenmenge des Ordners so klein wie möglich zu halten. Nun müssen Sie mit der rechten Maustaste auf diesen Ordner klicken und dann "Alle extrahieren..." auswählen. Nachdem dieser Prozess vollendet wurde, liegen zwei Dateien in dem gleichen Verzeichnis vor, zum einen die ZIP-Datei und ein normaler Ordner mit der gleichen Bezeichnung. Die ZIP-Datei wird von nun an nicht mehr benötigt und kann entfernt werden.
Wenn Sie nun in den normalen Ordner und dessen Unterordner gehen, sehen Sie zwei bis drei weitere Unterordner, die Sie ignorieren können, da diese die benötigten Dateien der Software beinhalten. Es wird empfohlen diese Dateien nicht zu manipulieren oder Werte umzuändern, da so Fehler entstehen könnten und die Software fehlerhaft beziehungsweise unbrauchbar gemacht wird. Es gibt zudem eine txt-Datei, in der Änderungen zu den verschiedenen Versionen dokumentiert werden und eine PDF-Datei mit den Lizenzbedingungen.
Die letzte Datei, ist die JAR- beziehungsweise EXE-Datei. Das ist die Datei, mit der die Software per Doppelklick gestartet werden kann. Allerdings erkennt hier ebenfalls das Betriebssystem, aus Sicherheitsgründen, die Software als Schadsoftware an. Unter Windows lösen Sie das Problem folgender Maßen:
Die interne Firewall von Windows wird beim Ausführen aktiv. Das heißt, der Start der Software wurde von dem Defender SmartScreen von Microsoft verhindert. Um dies zu umgehen, müssen Sie in dem neu geöffneten Fenster "Weitere Informationen" anklicken, damit unten ein zweiter Knopf sichtbar wird. "Trotzdem ausführen" kann nun angeklickt und so die Software gestartet werden. Nachdem nun einmal die Software gestartet wurde, wird diese Software kein weiteres Mal als Schadsoftware erkannt und kann ab sofort ganz normal gestartet werden.

Aktualisierung der Software

Wenn Sie eine neue Version der Software heruntergeladen haben und den Spielstand, beziehungsweise die Einstellungen beibehalten wollen, müssen Sie die JAR- beziehungsweise EXE-Datei und den bin-Ordner, mit den neuen Dateien überschreiben, also ersetzen. Die anderen Dateien dürfen nicht überschrieben werden, da sonst die Standardeinstellungen hergestellt werden. Wenn eventuell neue Einstellungen in einem Update hinzugefügt werden, dann gehen dennoch ihre gespeicherten Einstellungen verloren und Sie müssen die Software neu konfigurieren. Diese Änderungen können Sie in der mitgelieferten txt-Datei namens "Changelog" nachlesen. Außerdem werden Sie innerhalb der Software darauf hingewiesen.
